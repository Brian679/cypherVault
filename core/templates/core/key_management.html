Key Info

CipherVault uses RSA-2048 key pairs for:

  • Encryption: Receiver's public key encrypts the AES session key
  • Signatures: Sender's private key signs the file hash
  • Identity: Public keys establish sender/receiver identity

Key Rotation generates a new key pair and archives the old one. Files encrypted with old keys can still be decrypted using archived keys.
